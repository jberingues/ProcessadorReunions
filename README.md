# Processador de Reunions

Google Calendar + Transcripcions → Obsidian

## Setup Ràpid

```bash
# Instal·lar uv
curl -LsSf https://astral.sh/uv/install.sh | sh

# Setup
cd processador_reunions
uv sync

# Configurar (vegeu INICI_UV.md)
# Executar
uv run python src/reunio_interactiva.py
```

Documentació: `INICI_UV.md`
